# SKILL: 쉽게 설명하기

요청받은 코드나 개념을 아래 형식으로 설명한다.

## 출력 형식

```
## 한 줄 요약
- (초등학생도 알 만한 한 문장)

## 비유
- (일상 사물/상황에 빗대어)

## 핵심 3가지
1. ...
2. ...
3. ...
```

## 규칙
- 전문 용어를 쓰면 바로 옆에 괄호로 쉬운 말을 덧붙인다. 예: 캐시(자주 쓰는 걸 가까이 보관해두는 것)
- 핵심은 3개를 넘기지 않는다.
- 모르면 모른다고 말한다. 지어내지 않는다.
